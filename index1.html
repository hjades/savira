New Top-up Event

Project Context

Recognizing the unique preferences of Indonesian players, I led a comprehensive project to redesign our top-up event. The initiative was built on extensive research into local gaming behaviors, particularly the affinity for probability-based gameplay such as gacha games and spinning wheel mechanics.

Strategic Approach

The project addressed critical challenges including low player retention by developing a nuanced approach. We created a free-to-play mechanism that simultaneously educated players about the event and encouraged continued engagement. Critically, we ensured all user interfaces were localized in Indonesian to enhance accessibility and user experience.

Project Results

Player Participation Rate: 85%

Event Retention Rate Increase: 45%

Additional Impact: Attracted high-value players, resulting in increased daily revenue

Game User Interface Improvement

Initial Challenge

The existing game interface presented significant usability barriers, particularly for the elderly demographic that comprises a substantial portion of our user base. Small fonts, complex icons, and intricate design elements created substantial user experience challenges.

Design Strategy

Our redesign prioritized clarity, intuitive navigation, and demographic-specific usability. We systematically enhanced visual accessibility by enlarging interactive elements, increasing button sizes, and simplifying visual components. This approach demonstrated our commitment to creating inclusive, user-friendly design solutions.

Quantitative Outcomes

Average Icon Click-Through Rate: 30%

Side System Entry Rate Increase: 25%
